# iPhone 快捷指令配置指南

本文档介绍如何使用 iPhone "快捷指令" App 连接 Cloud Clip Lite，实现一键上传/下载剪切板内容。

---

## 前置准备

1. 在 Web 界面登录你的账号
2. 进入 **设备管理** 页面
3. 点击 **创建设备**，名称填 `iPhone`，类型选择 `iOS 快捷指令`
4. **复制生成的 API Token**（格式为 `cb_` 开头的字符串），仅显示一次

> 如果忘记复制，可以吊销后重新创建。

---

## 快捷指令 1：上传文本到剪切板

### 创建步骤

1. 打开 iPhone 上的 **快捷指令** App
2. 点击右上角 **+** 新建快捷指令
3. 点击底部 **搜索** 添加以下操作：

#### 步骤 1：获取剪贴板

- 搜索并添加 **"获取剪贴板"** 操作

#### 步骤 2：发送 HTTP 请求

- 搜索并添加 **"获取 URL 内容"** 操作
- 将方法改为 **POST**
- URL 填写：
  ```
  https://你的域名/api/clip
  ```
- 点击 **展开**（Show More）
- 请求头添加：
  | 键 | 值 |
  |---|---|
  | `Content-Type` | `application/json` |
  | `X-API-Token` | `cb_你的API_Token` |
- 请求体选择 **文件**，然后选择 **JSON**
- JSON 内容：
  ```json
  {
    "type": "text",
    "text": "剪贴板内容"
  }
  ```
  > 将 "剪贴板内容" 替换为上方 **"获取剪贴板"** 操作的输出变量（点击插入变量）

#### 步骤 3：显示通知（可选）

- 搜索并添加 **"显示通知"** 操作
- 标题：`已上传到剪切板`
- 内容：选择 HTTP 响应中的 `sha256` 字段

### 完整流程图

```
获取剪贴板 → POST /api/clip (X-API-Token) → 显示通知
```

## 快捷指令 2：下载最新剪切板

### 创建步骤

#### 步骤 1：发送 HTTP 请求

- 添加 **"获取 URL 内容"** 操作
- 方法：**GET**
- URL：
  ```
  https://你的域名/api/clip/latest
  ```
- 请求头：
  | 键 | 值 |
  |---|---|
  | `X-API-Token` | `cb_你的API_Token` |

#### 步骤 2：解析 JSON

- 添加 **"获取字典值"** 操作
- 从 HTTP 响应中获取键 `text` 的值

#### 步骤 3：复制到剪贴板

- 添加 **"拷贝至剪贴板"** 操作
- 内容选择步骤 2 的输出

### 完整流程图

```
GET /api/clip/latest → 提取 text → 拷贝至剪贴板
```

## 快捷指令 3：上传照片

### 创建步骤

#### 步骤 1：选择照片

- 搜索并添加 **"选择照片"** 操作
- 限制：单选

#### 步骤 2：发送 HTTP 请求

- 添加 **"获取 URL 内容"** 操作
- 方法：**POST**
- URL：
  ```
  https://你的域名/api/clip
  ```
- 请求头：
  | 键 | 值 |
  |---|---|
  | `X-API-Token` | `cb_你的API_Token` |
- 请求体选择 **文件**
- 文件选择步骤 1 的照片输出

## 添加到主屏幕 / 分享菜单

1. 在快捷指令详情页，点击 **分享** 按钮
2. 选择 **添加到主屏幕** — 可直接从桌面启动
3. 或选择 **在分享菜单中显示** — 可在任何 App 中分享内容时直接上传

---

## 配合自动化使用

1. 打开 **自动化** 标签页
2. 创建个人自动化：
   - **触发条件**：例如 "每天 08:00" 或 "到达特定位置"
   - **执行操作**：选择上述快捷指令

## 常见问题

### Q: 提示 401 Unauthorized
A: 检查 API Token 是否正确，是否已被吊销。到设备管理页面重新生成。

### Q: 提示 413 Request Entity Too Large
A: 文件超过单条上限（默认 10MB）。联系管理员调大 `max_item_size`。

### Q: 如何使用 HTTP 而非 HTTPS
A: 如果使用自签名证书或 HTTP，需要在 iPhone 设置 → 快捷指令 → 高级 → 开启 "允许不安全的连接"。

### Q: Token 安全吗
A: API Token 使用 SHA-256 哈希存储在服务端，明文仅在你创建时显示一次。每个 Token 绑定一个设备，可随时吊销。建议仅在个人设备上使用。
